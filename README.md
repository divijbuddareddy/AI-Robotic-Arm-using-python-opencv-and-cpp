This project implements a low‑cost AI‑controlled robotic arm that performs fully automatic color‑based object sorting by combining computer vision, embedded control, and optional IoT connectivity into one integrated system. A USB camera continuously captures video of the workspace, and a Python application processes each frame using OpenCV to convert it to HSV color space, apply predefined threshold ranges for red, green, and blue, clean the resulting masks with morphological operations, and detect sufficiently large contours corresponding to colored objects. For each detected object, the system classifies its color (optionally enhanced by a TensorFlow/Keras model trained on labeled color images for robustness to lighting and viewpoint changes) and sends a single‑character command (A, B, or C) over a serial link using PySerial to an Arduino Uno. On the embedded side, the project uses Arduino C++ running in the Arduino IDE to receive these commands and drive multiple DC motors via L293D motor drivers, executing fixed motion sequences that move the arm down to a pickup position, close the gripper, lift the object, rotate or translate the arm to the appropriate bin, open the gripper to release the object, and then return the arm to a home position, with behavior tuned through calibrated delay() timings for each color path. Hardware used includes the Arduino Uno microcontroller, several DC motors for base, arm joints, and gripper, L293D driver chips, a USB camera, and a dedicated power supply sized to handle both logic and motor loads; optionally, an ESP8266‑class Wi‑Fi module connects the system to an IoT platform such as Blynk or ThingSpeak to provide remote monitoring dashboards and simple control from a web or mobile interface. Overall, the project delivers an autonomous demo of AI‑driven industrial‑style sorting that showcases technologies such as Python, OpenCV, TensorFlow/Keras, PySerial, Arduino C++, DC motor control, L293D drivers, and IoT integration, while emphasizing modular design, configurability via HSV and timing settings, and educational value for students and hobbyists exploring intelligent robotics and automation.
