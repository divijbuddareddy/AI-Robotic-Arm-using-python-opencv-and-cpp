AI-Powered Robotic Arm with Color-Based Ball Detection
This project implements an AI-powered robotic arm system that uses computer vision to detect and respond to colored balls in real-time. The Python script leverages OpenCV and NumPy to capture video from a webcam, process each frame using HSV color space conversion, and detect red, green, and blue balls through advanced color thresholding, Gaussian blurring, and morphological operations to reduce noise. When a colored ball is detected, the system draws a bounding box around it, labels the color, and sends a corresponding single-character signal ('A' for red, 'B' for blue, 'C' for green) via serial communication to an Arduino board that controls the robotic arm's servos or actuators. The Arduino code receives these signals and triggers the appropriate arm movements or actions based on the detected ball color, with the system intelligently sending signals only when the detected color changes to avoid redundant transmissions and ensure smooth operation.

To use this AI-powered robotic arm project, you'll need Python with OpenCV, NumPy, and PySerial libraries installed, along with an Arduino board connected via USB (update the COM port in the Python script to match your system). The robotic arm should be assembled with servos connected to the appropriate Arduino pins, and a camera should be properly positioned to capture the workspace. After uploading the Arduino code to your board and calibrating the servos for your specific arm configuration, run the Python script and place colored balls in the camera's view. The AI vision system will automatically detect the ball colors, display bounding boxes with labels on the screen, and command the robotic arm to perform corresponding actions such as picking, sorting, or manipulating the detected objects. Press 'q' to exit the program. This project is perfect for learning AI robotics, automation, pick-and-place operations, color-based sorting systems, or building intelligent manufacturing and educational robotics applications.
